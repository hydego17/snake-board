# Snake Board

## Installation

### Clone this repo:

```sh
git clone https://github.com/hydego17/snake-board.git <your_folder_name>
```

### Install dependencies (React, React-DOM, Styled-Components):

```sh
npm install

# or

yarn
```

### Run this App:

```sh
npm run start

# or

yarn start
```

Go to http://localhost:3000/.
